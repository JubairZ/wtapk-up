  ## ✨ Features

  - 🌐 **Any website** → Native Android WebView app
  - 🎨 **12 built-in themes** (dark, light, colorful)
  - 🖥️ **10 splash screen templates**
  - 🔔 **Auto-update system** — checks GitHub for new versions
  - ✈️ **Telegram join dialog** — grow your community
  - 📡 **Offline banner** — animated, auto-hides when connection restores
  - 👆 **Swipe navigation** — back/forward with gesture
  - 📷 **QR Share** — share current page URL as QR code
  - 📋 **Copy URL** — one tap to clipboard
  - 🔠 **Text size control** — A+ / A- via JS injection
  - ⚙️ **In-app Settings** — analytics, cache, app info
  - 📊 **Local analytics** — page visits, session time (on-device)
  - 🤖 **GitHub Actions CI/CD** — auto-build & release APK
  - 🔐 **Optional keystore signing** — release-signed APK
  - 📥 **File download support**
  - 📤 **File upload / camera** support
  - 🚫 **Ad blocking** — block known ad domains
  - 💬 **FAB button** — WhatsApp / Telegram / Phone / Email
  - 🧩 **Custom JavaScript injection**
  - 🔒 **App expiry date** — set a deadline
  - ⭐ **Rating dialog** — ask users to rate
  - 🔴 **Exit confirmation dialog**
  - 🧾 **Navigation drawer** with custom links
  - 📱 **Bottom navigation bar** support

  ## 🔧 How to Use

  1. **Fork** the private repo `wtapk`
  2. Edit `lib/config/app_config.dart` — set your URL, name, theme, etc.
  3. Push to GitHub → Run **"Build & Release APK"** action
  4. APK is automatically released here in this public repo!
